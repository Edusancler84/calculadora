# calculadora simples
Projeto de um calculadora simples

## 🚀 Funcionalidades

- Adição ➕
- Subtração ➖
- Multiplicação ✖️
- Divisão ➗
- Limpar resultado
- Interface simples e responsiva

# Tecnoligias usadas 
- **HTML5** → Estrutura da calculadora  
- **CSS3** → Estilização e layout  
- **JavaScript** → Lógica das operações
